1. O que é o Balsamiq Wireframes? Qual estilo de prototipação ele utiliza?

O Balsamiq Wireframes é uma ferramenta de software utilizada para criar wireframes de interface de usuário de forma rápida e eficiente. O Balsamiq Wireframes utiliza o estilo de prototipação de baixa fidelidade.

2. O que é o figma? Qual estilo de prototipação ele utiliza?

O Figma é uma ferramenta de design colaborativo baseada na web, utilizada principalmente para criar interfaces de usuário, protótipos interativos e colaboração em tempo real entre equipes de design e desenvolvimento. Ele usa prototipação de alta e baixa qualidade.

3. Comente sobre um problema da sociedade que possa ser resolvido por um sistema computacional e porque resolver tal problema é importante.

Um problema é Gestão inadequada de resíduos sólidos urbanos e Um sistema computacional pode melhorar o planejamento e otimização de rotas para otimizar as rotas de coleta, levando em consideração fatores como localização dos pontos de coleta, tráfego e capacidade dos veículos.
